# Synth: Odyssey

Digital homage to the ARP Odyssey synthesizer.

![screenshot](./screenshot.png)

## Building

The project is built using CMake and is split into libraries.

### Dependencies

On Fedora Linux, install the following dependencies for each module:

| Module | Dependencies   |
|--------|----------------|
| Alsa   | alsa-lib-devel |
| Imgui  | glfw-devel     |
| Qt     | qt5-devel      |
